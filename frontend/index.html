Building the infrastructure that powers the internet

I'm a Cloud and DevOps Engineer with a strong security operations background, passionate about designing secure, automated, and resilient cloud systems. I bridge the gap between DevOps, SOC, and DevSecOps practices to deliver security-first, production-ready solutions.

I work across AWS, Azure, and GCP, using Infrastructure as Code (Terraform, CloudFormation) and CI/CD pipelines (GitHub Actions, Jenkins, AWS CodePipeline) to automate consistent, reliable deployments. My SOC background means security is never an afterthought — it's baked into every pipeline and architecture decision.

I also bring a frontend development background, giving me end-to-end understanding of application behavior, deployment pipelines, and application-layer security risks.
